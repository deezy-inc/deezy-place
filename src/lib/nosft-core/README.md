# nosft-core (vendored)

Vendored copy of [deezy-inc/nosft-core](https://github.com/deezy-inc/nosft-core)
(main branch, July 2026), transpiled from TypeScript to plain JS. The npm
package is deprecated; this app now owns this code. Entry point is `index.js`
(`Nosft`), consumed by `src/services/nosft.js`.
